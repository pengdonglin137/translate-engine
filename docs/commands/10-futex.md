# Linux Kernel - Futex

## 概述
- 快速用户空间互斥锁（Fast Userspace muTEX）
- 结合用户空间和内核空间的同步机制
- 无竞争时纯用户空间操作，避免系统调用开销

## 工作原理
1. 初始化：在用户空间设置锁变量
2. 加锁：尝试原子操作，失败则进入内核等待
3. 解锁：原子操作，必要时唤醒等待者

## Futex操作
- FUTEX_WAIT：等待锁变为指定值
- FUTEX_WAKE：唤醒等待的线程
- FUTEX_REQUEUE：将等待者移到另一个futex
- FUTEX_CMP_REQUEUE：比较并重新排队

## 应用场景
- pthread互斥锁
- 读写锁
- 条件变量
- 信号量

## 性能特点
- 无竞争：纯用户空间，极快
- 有竞争：系统调用开销，但仍比纯内核锁快
- 适合大多数同步场景
