# Linux Kernel - 同步原语

## 自旋锁
- spinlock：忙等待锁，适用于短临界区
- 读写锁：允许多个读取者，独占写入者
- 顺序锁：无锁读取，适合读多写少

## 信号量
- semaphore：可睡眠的锁，适用于长临界区
- 互斥锁（mutex）：简化版信号量
- 读写信号量：允许多个读取者

## RCU（读-拷贝-更新）
- 无锁同步机制
- 读取者无需等待
- 写入者创建副本，更新后原子替换
- 适用于读多写少的场景

## 原子操作
- 原子读写
- 原子比较并交换（CAS）
- 原子加减
- 内存屏障

## 内存屏障
- 读内存屏障（rmb）
- 写内存屏障（wmb）
- 全内存屏障（mb）
- 确保内存操作顺序

## 每CPU变量
- 避免锁竞争
- 用于统计计数器
- 热路径性能优化
